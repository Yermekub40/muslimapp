# 🕌 Салих — Инструкция по установке PWA

## Структура файлов

```
salih/
├── index.html          ← главный файл (salih-app-final.html → переименовать)
├── manifest.json       ← PWA манифест
├── service-worker.js   ← офлайн + уведомления
└── icons/
    ├── icon-72.png
    ├── icon-96.png
    ├── icon-128.png
    ├── icon-144.png
    ├── icon-152.png
    ├── icon-192.png
    ├── icon-384.png
    └── icon-512.png
```

---

## Шаг 1 — Создание иконок

1. Откройте файл `generate-icons.html` в браузере
2. Нажмите **"Скачать все иконки"**
3. Создайте папку `icons/` и поместите туда все скачанные PNG файлы

---

## Шаг 2 — Переименование

Переименуйте `salih-app-final.html` → `index.html`

---

## Шаг 3 — Хостинг (выберите один)

### Вариант A: GitHub Pages (бесплатно)

1. Зарегистрируйтесь на [github.com](https://github.com)
2. Создайте новый репозиторий (например: `salih-app`)
3. Загрузите все файлы:
   ```
   index.html
   manifest.json
   service-worker.js
   icons/icon-72.png
   icons/icon-96.png
   ... (все иконки)
   ```
4. Перейдите в **Settings → Pages**
5. В разделе "Source" выберите **main branch**
6. Ваш сайт: `https://ваш-логин.github.io/salih-app`

### Вариант B: Netlify (бесплатно, проще)

1. Зайдите на [netlify.com](https://netlify.com)
2. Нажмите **"Deploy manually"**
3. Перетащите папку `salih/` прямо на страницу
4. Готово! Получите адрес вида `https://xyz123.netlify.app`

### Вариант C: Vercel (бесплатно)

1. Зайдите на [vercel.com](https://vercel.com)
2. Импортируйте с GitHub или загрузите папку
3. Готово!

---

## Шаг 4 — Установка на телефон

### Android (Chrome)
1. Откройте сайт в Chrome
2. Нажмите **⋮ → Добавить на главный экран**
3. Или дождитесь баннера "Установить Салих" внизу экрана

### iOS (Safari)
1. Откройте сайт в Safari
2. Нажмите **Поделиться → На экран «Домой»**
3. Нажмите **Добавить**

---

## Шаг 5 — Уведомления

При первом открытии приложение запросит разрешение на уведомления.
- **Разрешите** — получите уведомления о намазе, азкарах и вирде
- На **Android** уведомления работают в фоне
- На **iOS** — только когда приложение открыто (ограничение Apple)

---

## Офлайн режим

После первого открытия приложение работает **без интернета**:
- Все времена намазов доступны офлайн
- Азкары, календарь, пост, вирд — всё работает
- Данные сохраняются в localStorage телефона

---

## Часто задаваемые вопросы

**Q: Нужно ли платить за хостинг?**
A: Нет. GitHub Pages, Netlify и Vercel — бесплатны для личного использования.

**Q: Приложение появится в Play Market / App Store?**
A: Нет. PWA устанавливается напрямую через браузер, без магазинов.

**Q: Данные сохраняются?**
A: Да, все данные (настройки намазов, прогресс азкаров, вирд) хранятся в памяти телефона.

**Q: Как обновить приложение?**
A: Просто обновите файлы на хостинге — при следующем открытии Service Worker автоматически скачает новую версию.
